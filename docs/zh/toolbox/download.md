# 下载与安装

## 下载方式

1. 从 Obsidian 插件市场下载 （😜暂不支持）
   - 打开 Obsidian
   - 进入设置 > 社区插件
   - 搜索 "Obsidian Toolbox Plugin"
   - 点击安装

2. 从 GitHub 下载 （✅推荐）
   - 访问 [GitHub 仓库](https://github.com/TracingOrigins/obsidian-toolbox-plugin)
   - 下载最新版本
   - 解压到 Obsidian 插件目录

## 安装步骤

1. 从 Obsidian 插件市场安装 （😜暂不支持）
   - 打开 Obsidian
   - 进入设置 > 社区插件
   - 搜索 "Obsidian Toolbox Plugin"
   - 点击安装
   - 启用插件

2. 手动安装 （✅推荐）
   - 下载插件
   - 解压到 Obsidian 插件目录
   - 重启 Obsidian
   - 启用插件

## 配置说明

1. 基本配置
   - 启用/禁用工具
   - 配置工具顺序
   - 配置工具视图

2. 工具配置
   - 自动时间戳工具
   - 复制标签页链接工具
   - 复制文件路径工具
   - 行内代码复制工具
   - 文档属性排序工具

## 注意事项

- 确保 Obsidian 版本 >= 1.0.0
- 确保插件版本与 Obsidian 版本兼容
- 确保插件目录有写入权限
- 确保插件配置正确
